# 预备

## 机器学习中的数学基础 
机器学习需要俱备一定的数学基础， 主要指的是：

- 多元微积分
- 线性代数
- 概论统计
- 优化理论

## 计算机科学基础

- 数据结构与算法基础
- Python程序设计
